## Installation

#### Front-End:

Israel-News required [Angular] V11 to run. Runs on PORT 4200.

```sh
$ npm install
```

**or:**

```sh
$ yarn install
```

#### Back-End:

Israel-News required [Node.js] V12.18.03 to run. Runs on PORT 3000 (for changing
you need to set in the back and the front).

```sh
$ npm install
```

**or:**

```sh
$ yarn install
```

## Activated

#### Front-End:

```sh
$ npm start
```

**or:**

```sh
$ yarn start
```

#### Back-End:

- run with nodemon.

```sh
$ npm run dev
```

**or:**

```sh
$ yarn run dev
```

- **run without nodemon.**

```sh
$ npm run start
```

**or:**

```sh
$ yarn run start
```
